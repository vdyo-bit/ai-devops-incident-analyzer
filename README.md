# AI-Assisted Incident Diagnosis Platform

## Goal
Learn Linux, observability, and SRE-style incident analysis by building
an AI-assisted diagnosis system.

## What this project is NOT
- Not auto-remediation
- Not AI-controlled infrastructure

## Learning focus
- Linux internals
- Metrics & logs
- Failure analysis
- Safe AI usage

## AI Safety & Reliability Principles
- AI is used only for analysis, never for remediation
- AI outputs are advisory, not authoritative
- Uncertainty is explicitly allowed and documented
- Final decisions are made using first-principles reasoning
